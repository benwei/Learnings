# Introducton
  This is a experiments for GNUStep and GNU objc

# Setup in Ubuntu

 - I only tested it on Ubuntu 10.10.

## Install gobjc

 - sudo apt-get install gobjc

## Install gnustep

 - sudo apt-get install gnustep
 - sudo apt-get install gnustep-devel

# Build procedure

 - cd hello-objc-linux 
 - make 

# Testing
 - cd hello-objc-linux
 - make run
